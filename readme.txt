== Description ==
 
> Important Recommendations : For better user experience we have shifted  **Stack Slider 3d Image Slider** plugin in our to new plugin ie [SlidersPack – All In One Image/Post Slider](https://wordpress.org/plugins/sliderspack-all-in-one-image-sliders/). It help us to maintain our all slider plugins in a single plugin package and provide you better and fast support.  
 
Check [FREE DEMO](http://demo.wponlinesupport.com/sliderspack-all-in-one-image-post-slider/) 
Check [PRO DEMO and Features](http://demo.wponlinesupport.com/prodemo/sliderspack-pro-all-in-one-image-post-slider/) for additional information.

Stack Slider is a super lightweight jQuery plugin used for generating an image carousel slider that acts like a 3D rotator with navigation and pagination. 

= This plugin contain shortcode: =

<code>[stack_slider id="XX"]</code>

= Here is Template code =

<code><?php echo do_shortcode('[stack_slider id="XX"]'); ?></code>

= Features include =

* Also work with Gutenberg shortcode block.
* Fully responsive
* Easy to install
* Given shortcode and template code.
* Works on modern browsers which have CSS3 transform and transition support.

= Privacy & Policy =
* We have also opt-in e-mail selection , once you download the plugin , so that we can inform you and nurture you about products and its features.


== Installation ==
1. Upload the 'stack-slider-a-3d-imageslider' folder to the '/wp-content/plugins/' directory.
2. Activate the "stack-slider-a-3d-imageslider" list plugin through the 'Plugins' menu in WordPress.
3. Use the shortcode 
<code>[stack_slider id="XX"]</code>
